# nginx installation
- installing nginx in **ubuntu vm**
```
apt install nginx 
```
- installing nginx in **redhat vm**
```
yum install nginx
``
- to check the status of nginx 
```
systemctl status nginx 
systecmt enable nginx 
```
NOTE : 
  document root path of nginx is /usr/share/nginx/html
